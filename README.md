# pol

## Getting Started

## Installation

- Install `react-native` first

```bash
npm i react-native -g
yarn add pol
mkdir Example
cd Example
yarn install #(or with npm)
npm start
```


### Example

or clone the repo and play with the example project

```bash
$ git clone https://github.com/garrettmac/pol
$ cd pol/Example
$ yarn install
$ npm start
```
### Basic Usage

- In your `index.js`, use:
```bash
import X from "pol"
........
```
